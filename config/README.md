# config · 配置

世界杯预测里**需要人工定下来**的配置(既不是代码,也不是采集来的数据)。
这些和 [`../PLAN.md`](../PLAN.md) 的「待定决策」**直接挂钩**——方案里拍板什么,就固化到这里。

## 预期内容(随 PLAN 决策落地,现在多为空)

| 文件 | 是什么 | 绑定的 PLAN 决策 |
|---|---|---|
| `teams.json` | 48 队清单(队名 + 分组 + FIFA 排名) | —— |
| `fixtures.json` | 官方赛程 + 滚动切点表 | ② 切点表 |
| `keywords_template.json` | 每队关键词模板(采集用) | —— |
| `sources.json` | 赛果 / 赔率数据源 + API 配置 | ③ 数据源 |
| `platforms.json` | media 平台清单(中文7 / 加国际) | ④ 平台范围 |
| `.env.example` | API key 模板(真 key 不进 git) | —— |

> 这些定下来,采集和滚动代码才能跑。**先在 `PLAN.md` 拍板四个决策,再固化到这里。**
